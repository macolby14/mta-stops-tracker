##MTA_FLASK

-   backend for mta_py app
-   Flask server that calls mta api to provide api for frontend


## Building
- These steps are based on the [Python Packing guide](https://packaging.python.org/en/latest/key_projects/#build)
- Install dependencies with `python -m pip install .` This installs from the pyproject.toml
- Build a wheel for distribution with `python -m build`.
